# Financial-Health-Dashboard
 ## Overview
The Financial Health Report Dashboard is an interactive Power BI dashboard designed to analyze an individual's or organization's financial performance. The dashboard provides insights into income, expenses, savings, debt, investments, and net worth, helping users understand their overall financial health.

## Objectives
Monitor financial performance through key metrics.
Compare income and expenses over time.
Analyze savings trends and investment patterns.
Evaluate debt distribution across occupations.
Compare assets and liabilities across different cities.
Support financial decision-making through visual analytics.

## Tools Used
Power BI Desktop
DAX (Data Analysis Expressions)
Data Modeling
Data Visualization

## Key Insights
Tracks overall financial health using multiple KPIs.
Identifies spending behavior and savings patterns.
Compares financial performance across cities.
Evaluates debt burden and net worth position.
Supports data-driven financial planning.
